## 📌 프로젝트 1 — 자동화 도구 비교 구현: "출근길 날씨 요정"

### 1. 워크플로우 정의

- **목적:** 매일 아침 07:00에 날씨를 확인하여 비 예보 시 우산 지참 알림 발송
- **구조:** `Trigger(Schedule)` → `Action(Weather API)` → `Filter(조건 분기)` → `Action(Gmail)`

---

### 2. 워크플로우 설계

```
Trigger      : Schedule (매일 오전 07:00)
Action 1     : OpenWeatherMap (날씨 데이터 조회)
Router       : 조건 분기 (비 오는 날 vs 맑은 날)
  ├─ Filter 1 (비 오는 날)
  │    └─ Action 2 (비 오는 날 알림)
  └─ Filter 2 (맑은 날)
       └─ Action 3 (맑은 날 알림 / 선택)
```

---

### 3. 비교 분석표

| 비교 항목 | Make | Zapier |
|---|---|---|
| UI/UX | 시각적 노드(Node) 방식 | 리스트(List) 수직 방식 |
| 조건 분기 | Router (무료 범위 내 강력) | Router (무료 범위 내 강력) |
| 설정 난이도 | 초기 학습 곡선 있음 | 매우 직관적이고 쉬움 |
| 무료 플랜 | 1,000 Ops/월 | 100 Tasks/월 |
| 실행 로그 | 상세 데이터 흐름 확인 가능 | 직관적인 히스토리 관리 |

---

### 4. 종합 의견

**장단점 비교**

- **Make:** 조건 분기 설계가 자유롭고 무료 효율이 좋으나 초기 설정이 다소 복잡함
- **Zapier:** 연결 편의성이 뛰어나지만 조건 분기 등 핵심 기능이 유료인 경우가 많음

**추천 상황**

- 복잡한 조건 처리가 필요한 개인 자동화 프로젝트 → **Make**
- 단순하고 빠른 연동이 필요한 업무 → **Zapier**

---

### 5. 용어 정의

| 용어 | 설명 | 예시 |
|---|---|---|
| **Trigger** | 자동화의 시작 신호. 사용자가 설정한 특정 이벤트가 감지되면 자동화가 즉시 실행됨 | 매일 오전 7시, 구글 폼 제출, 새 이메일 도착 |
| **Action** | 트리거 발생 후 시스템이 수행하는 실제 작업 | 이메일 발송, 구글 시트 행 추가, 메신저 알림 |
| **Filter** | 특정 조건이 참(True)일 때만 다음 단계로 진행하게 하는 조건문 | 강수 확률이 60% 이상일 때만 다음 단계 진행 |
| **Route** | 하나의 트리거에서 시작해 데이터 조건에 따라 여러 경로로 분기하는 역할 | 비 → '우산 알림' 경로 / 맑음 → '기분 좋은 인사' 경로 |

## 📌 프로젝트 2 — 자유 주제: "오늘의 캘린더 일정 리포트 자동 발송 서비스"

> **자동화 목표:** 매일 아침 구글 캘린더 일정을 자동으로 조회하여, 중요 일정은 즉시 개별 알림을 보내고 전체 일정은 통합 리포트 이메일로 수신

---

### 1. 업무 정의 및 선정 이유

**업무 흐름**

```
구글 캘린더 일정 검색
  └─ 라우터 (조건 분기)
       ├─ 경로 1: 중요 일정 → 개별 이메일 발송
       └─ 경로 2: 전체 일정 → 텍스트 취합 → 통합 리포트 이메일 발송
```

**선정 이유**

매일 아침 흩어져 있는 일정을 일일이 확인하는 시간을 단축하고, 중요한 일정은 놓치지 않으면서 전체적인 하루 흐름을 한눈에 파악하기 위함

---

### 2. 워크플로우 설계

```
Trigger    : Google Calendar – Search Events (매일 지정 시간)
Router     : 일정의 중요도에 따른 조건 분기 설정
  ├─ Action 1 (개별 알림)
  │    └─ Gmail – Send an email (중요 일정 발생 시 즉시 발송)
  └─ Action 2 (통합 리포트)
       ├─ Tools – Text Aggregator (모든 일정을 텍스트로 취합)
       └─ Gmail – Send an email (전체 일정 통합 메일 발송)
```
